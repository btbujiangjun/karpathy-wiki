A 27B parameter code language model specialized for web game creation. Trained through a 3-stage pipeline: Continue Pretraining → SFT → execution-grounded RL. Part of the OpenGame framework for end-to-end agentic game coding.

## Key Details

- Size: 27B parameters
- Domain: Web game code generation
- Training: CPT + SFT + execution-grounded RL
- Evaluation: OpenGame-Bench (Build Health, Visual Usability, Intent Alignment)
- SOTA on 150 diverse game prompts
