## Problem

Classical game AI taxonomies (Shannon's classification) predate LLMs and do not account for modern language model capabilities.

## Method

Extends Shannon's taxonomy of game-playing machines using LLMs. Four game classes: dictionary-based, rigorously solvable, heuristic-based, learning-based. Builds an interactive agentic engineering environment (Nemobot) for users to create and deploy LLM-powered game agents with self-programming AI via crowdsourced learning.

## Key Results

- LLM chatbot demonstrates capabilities across all four game classes
- Interactive environment enables non-expert game agent creation

## Key Innovations

- Modernizes Shannon's taxonomy with LLM capabilities
- Interactive agentic engineering environment
- Self-programming AI via crowdsourced learning from human interactions
