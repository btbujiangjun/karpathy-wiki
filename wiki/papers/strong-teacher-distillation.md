## 问题背景

在 LLM 预训练中，Knowledge Distillation（Knowledge Distillation）通常遵循一个直觉假设：教师模型越强，学生模型学到的效果越好。因此研究工作集中在构建更强的教师模型上，但这带来了巨大的计算开销。本文挑战了这一假设，系统研究了"弱教师能否训练出好学生"以及"强教师是否总优于弱教师"这两个核心问题。

## 方法详述

本文在标准autoregressive语言模型预训练框架中引入蒸馏损失（KD loss），与原始的 LM loss（next token prediction）联合优化：

```
L_total = L_lm + λ · L_kd
```

其中 L_kd 是学生与教师输出分布之间的 KL 散度。关键实验变量包括：
- **教师大小**：从小型（125M）到大型（1.5B）多个规模
- **教师训练程度**：从欠训练到完全收敛的不同 checkpoint
- **蒸馏权重 λ**：不同损失混合比例
- **学生大小**：从 350M 到 1.3B

## 主要创新点

- **推翻了"强教师更好"的直觉假设**：实验发现教师并非越强越好，过强的教师会导致学生overfitting教师的偏见
- **发现非单调的蒸馏行为**：当教师能力超过某个阈值后，学生性能反而下降——存在最优教师强度
- **弱到强蒸馏可行**：适当混合 LM loss 和 KD loss 后，弱/欠训练教师也能有效提升学生性能

## 实验结果与对比

| 教师配置 | 学生 (1.3B) 下游平均分 | 与无蒸馏基线对比 |
|----------|----------------------|----------------|
| 无蒸馏（基线） | 62.3 | - |
| 弱教师 (125M, 欠训练) | **64.1** | **+1.8** |
| 中等教师 (350M, 充分训练) | **64.8** | **+2.5** |
| 强教师 (1.5B, 充分训练) | 63.2 | +0.9 |
| 最强教师 (1.5B, 过训练) | 61.8 | **-0.5** (倒退) |

**关键发现：**
- 中等教师（350M 充分训练）对学生提升最大（+2.5）
- 最强教师（1.5B 过训练）反而导致学生性能下降（-0.5）
- 弱到强蒸馏（125M 欠训练 → 1.3B）依然有效（+1.8），说明蒸馏不是大教师的专利
- OOD 和下游泛化的提升幅度大于域内拟合，说明蒸馏主要改善的是泛化能力而非记忆能力

**核心结论：** 在 LLM 预训练蒸馏中，"够用就好"的教师比"最强"的教师更有效。最佳策略是用一个中等规模、充分训练但不过训练的教师，搭配合适的 KL 权重。

## 局限性

- 实验规模上限为 1.5B 教师 / 1.3B 学生，更大规模（10B+）下结论是否成立有待验证
- 未深入分析为什么强教师会导致性能下降（推测是overfitting的偏见被蒸馏传递）
- 蒸馏带来的提升幅度（+1~2.5 分）相对有限，需要权衡额外计算开销
